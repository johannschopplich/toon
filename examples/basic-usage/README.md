# Basic Usage Examples

Learn TOON fundamentals with these simple examples.

## Files in this directory

- **[simple-object.js](./simple-object.js)** - Basic object encoding/decoding
- **[arrays.js](./arrays.js)** - Working with different array types
- **[nested-data.js](./nested-data.js)** - Handling nested objects and arrays
- **[options.js](./options.js)** - Using encoding options (delimiters, indentation)
- **[type-handling.js](./type-handling.js)** - How TOON handles different data types

## Running the examples

```bash
# Install dependencies first
npm install @toon-format/toon

# Run any example
node simple-object.js
node arrays.js
# ... etc
```

## What you'll learn

1. **Basic encoding/decoding** - Convert between JSON and TOON
2. **Array formats** - When TOON uses tabular vs list format
3. **Token efficiency** - See actual token savings
4. **Configuration options** - Customize output format
5. **Type conversions** - How different JS types are handled
